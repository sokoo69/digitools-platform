# DigiTools Platform

I built DigiTools as a simple place to find and buy digital tools things like resume builders, design tools, and portfolio makers. It's put together with React and Tailwind CSS, and I tried to keep the experience straightforward and fast.

## Built With

- React.js - UI
- Tailwind CSS - Styling
- DaisyUI - Component library
- React-Toastify - Notifications
- Vite - Build tool

## Features

### Shopping Cart
You can add products, remove them, and checkout — pretty standard stuff. The navbar updates as you go, and there's a small toast notification on every action so nothing feels like it happened silently.

### Product Cards
There are 6 digital tools on the page, each showing the price, a few key features, and a tag (Popular, New, or Best Seller). Once you add something to your cart, the button reflects that too.

### Responsive
Works on mobile, tablet, and desktop. Nothing fancy — it just doesn't break on smaller screens.
